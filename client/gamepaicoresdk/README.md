# Build SDK

## Install gomobile

    go get golang.org/x/mobile/cmd/gomobile
    gomobile init

## For Android

    gomobile bind -target=android gamecenter.mobi/paicode/client/gamepaicoresdk

## For iOS

    gomobile bind -target=ios gamecenter.mobi/paicode/client/gamepaicoresdk
